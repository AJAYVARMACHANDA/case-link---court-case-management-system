# 🔗 CaseLink – Smart Case Management System

## 📌 Overview

**CaseLink** is a web-based case management system designed to streamline the process of creating, tracking, and managing cases efficiently.
It helps organizations maintain structured records, monitor case progress, and improve overall workflow transparency.

---

## 🚀 Features

* 🔐 Secure User Authentication (Admin/User roles)
* 📁 Create, update, and manage cases
* 📊 Dashboard for case tracking and insights
* 🔄 Real-time status updates (Open, In Progress, Closed)
* 🔍 Search and filter cases
* 🗂️ Organized data storage for easy access
* 👨‍💼 Role-based access control

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Database:** MongoDB
* **Frontend:** HTML, CSS, JavaScript
* **Tools:** Git, Postman

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ajayvarmachanda/court-case-management-system
cd caselink-project
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
python app.py
```

### 5️⃣ Open in Browser

```
http://127.0.0.1:5000/
```
